**Must-Haves:**

### Platform Comparison: API/Integration Capabilities

#### **OPTION 1: Custom LMS (Canvas or Moodle) + Maxi Chatbot (Separate)**

**Architecture:**
- Canvas/Moodle: Course content, assignments, grading, discussions
- Bitcoinsingularity.ai: Separate web app with Maxi chatbot (custom-built)
- Integration: Students get link to chatbot from Canvas, authenticate via SSO

**Canvas API Capabilities:**
- ✅ Robust REST API (read/write course data, assignments, discussions)
- ✅ Single Sign-On (LTI, OAuth, SAML)
- ✅ Webhooks (trigger Maxi when student submits assignment, posts discussion)
- ✅ Custom navigation links (embed bitcoinsingularity.ai in Canvas sidebar)
- ✅ Grade passback (Maxi can submit grades for participation)

**Moodle API Capabilities:**
- ✅ REST API (similar to Canvas)
- ✅ Open-source (more customization flexibility than Canvas)
- ✅ Plugin architecture (could build native Maxi plugin)
- ⚠️ More technical setup required (self-hosted or paid hosting)

**Custom Chatbot (bitcoinsingularity.ai):**
- Built with: React/Next.js frontend + OpenClaw backend + Lightning Network integration
- Hosted: Boyd's existing infrastructure (mining rig or separate server)
- Authentication: OAuth with Canvas/Moodle (students log in once, access both)
- Data sync: Maxi pulls student info, course progress, assignment submissions from LMS via API

**Pros:**
- ✅ Maximum flexibility (Maxi can do anything—no platform limitations)
- ✅ Full control over chatbot UX/features
- ✅ Lightning Network integration (Maxi can send/receive sats natively)
- ✅ Scales independently (if Maxi gets popular, doesn't affect LMS performance)

**Cons:**
- ❌ More technical complexity (two systems to maintain)
- ❌ Higher setup cost ($5K-10K custom dev)
- ❌ Authentication flow less seamless (students click link to go to chatbot)

**Cost:**
- Canvas: $10K-20K/year (enterprise)
- Moodle: $2K-5K/year (hosting) OR $0 (self-hosted)
- Custom chatbot dev: $5K-10K (one-time)
- **Total Year 1: $15K-30K**

---

#### **OPTION 2: Maven + Maxi Chatbot Integration**

**Architecture:**
- Maven: Course platform (live sessions, discussions, assignments)
- Bitcoinsingularity.ai: Custom chatbot (iframe embedded in Maven OR separate link)

**Maven API Capabilities:**
- ⚠️ **Limited API** (Maven is closed platform, minimal public API)
- ✅ Zapier integration (can trigger webhooks on events: new student enrolled, assignment submitted)
- ⚠️ No SSO (students would need separate login for chatbot)
- ✅ Iframe embeds (can embed external tool, but authentication tricky)

**Workaround:**
- Students access Maxi via external link (bitcoinsingularity.ai)
- Authentication: Students enter Maven email + course code → Maxi verifies via Zapier webhook
- Not seamless, but functional

**Pros:**
- ✅ Faster Maven setup (2-4 weeks vs 6-8 weeks Canvas)
- ✅ Lower Maven cost ($12K-18K/year vs $10K-20K Canvas)
- ✅ Cohort features (live sessions, community built-in)

**Cons:**
- ❌ Weak API (hard to deeply integrate Maxi)
- ❌ Authentication not seamless (students manage two logins)
- ❌ Less future-proof (if Boyd scales to corporate clients, Maven limitations become problem)

**Cost:**
- Maven: $12K-18K/year
- Custom chatbot dev: $5K-10K (one-time)
- **Total Year 1: $17K-28K**

**Verdict:** Maven works for Year 1 (fast launch), but migrate to Canvas Year 2 for better integration.

---

#### **OPTION 3: Kajabi + Maxi Chatbot Integration**

**Architecture:**
- Kajabi: All-in-one platform (courses, funnels, email, community)
- Bitcoinsingularity.ai: Custom chatbot (iframe embedded OR separate link)

**Kajabi API Capabilities:**
- ✅ REST API (read/write members, purchases, course progress)
- ✅ Webhooks (trigger Maxi on events: member enrolled, lesson completed)
- ✅ Custom code (HTML/CSS/JS in course builder—can embed chatbot iframe)
- ⚠️ SSO limited (OAuth possible but not native)

**Integration Path:**
- Embed Maxi chatbot as iframe in Kajabi course pages (students see chatbot sidebar while watching lectures)
- Authentication: Pass student email from Kajabi to chatbot via URL parameter (simple but functional)
- Webhooks: Maxi receives notifications when student completes lessons, submits assignments

**Pros:**
- ✅ All-in-one (less complexity than Canvas + custom)
- ✅ Good API (better than Maven, almost as good as Canvas)
- ✅ Branded mobile app (students access course + chatbot on phone)
- ✅ Marketing tools (funnels, email) built-in

**Cons:**
- ❌ Higher cost ($4,788/year Kajabi Pro + custom dev)
- ❌ Less academic feel (more "online course creator" than "Executive MBA")

**Cost:**
- Kajabi Pro: $4,788/year
- Custom chatbot dev: $5K-10K (one-time)
- **Total Year 1: $10K-15K**

**Verdict:** Good middle ground (cheaper than Canvas, better API than Maven).

#### **OPTION 4: Full Custom Platform (No LMS)**

**Architecture:**
- Build everything from scratch: Course pages, video hosting, discussions, assignments, chatbot (all on bitcoinsingularity.ai)

**Pros:**

- ✅ Scalable (can handle 1,000+ students without platform limits)

**Cons:**
- ❌ ❌ **Massive dev effort** ($20K-50K one-time + ongoing maintenance)
- ❌ Longer timeline (3-6 months to build vs 2-4 weeks with existing platform)
- ❌ Boyd becomes "software company" (needs dev team or contractor)

**Cost:**
- Custom dev: $20K-50K (one-time)
- Hosting: $500-2K/year (AWS/Vercel)

- **Total Year 1: $25K-60K**

### **RECOMMENDED PLATFORM STRATEGY**

**Year 1 (Fast Launch): Maven + Maxi Chatbot (Separate Link)**
- Why: Fastest to market (2-4 weeks), lowest setup cost ($17K-28K)
- Trade-off: Weak integration (students click link to access Maxi, separate login)
- MVP approach: Validate market, iterate on Maxi features, collect feedback

**Year 2 (Scale + Optimize): Migrate to Canvas + Maxi Chatbot (Integrated)**
- Why: Robust API, SSO, corporate-grade (enterprise clients need Canvas credibility)
- Investment: Use Year 1 revenue ($700K+) to fund Canvas setup + deeper Maxi integration
- Result: Seamless UX, Maxi embedded in course sidebar, single login

**Year 3+ (Optional): Full Custom Platform**
- Why: If scaling to 500+ students OR selling platform as white-label product to other educators
- Investment: $50K-100K (fund from Year 1-2 profits)
