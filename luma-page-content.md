## FULL DESCRIPTION

**AI agents are gaining the ability to act and transact autonomously. The payment rails they need don't exist yet.**

Multiple competing standards are emerging simultaneously:
- **L402** — Lightning-native payments for AI agents
- **x402** — Coinbase's cross-chain agent payment protocol  
- **ACP** — The Agentic Commerce Protocol (Stripe × OpenAI)
- **AP2** — Google's Agent Payment Protocol

Each takes a fundamentally different approach to the same question: *How should autonomous software pay for services, data, compute, and goods — without human intervention, without fraud exposure, and without requiring identity?*

**No one has brought these teams together. Until now.**

The Open Agentic Payments Summit is a 3-hour live virtual event convening protocol designers, product leads, and builders for an honest, technical conversation about what's being built, what's working, and what's missing.

This is not a pitch event. It's a working forum.
