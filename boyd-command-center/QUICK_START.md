## 🎨 Understanding the Dashboard

### Color System:
- 🔴 **RED** = <24 hours left (DO NOW!)
- 🟡 **YELLOW** = 24-72 hours (PLAN SOON)
- 🟢 **GREEN** = >72 hours (ON TRACK)

### Status Badges:
- **ACTIVE** (blue, pulsing) = Being worked on now
- **QUEUED** (gray) = Up next, not started
- **COMPLETE** (green) = Done!

### Progress Bars:
- Empty ░░░░░ = 0% (just started)
- Half ████░░░░ = 50% (halfway)
- Full ████████ = 100% (complete)

## 🆘 Troubleshooting (Quick Fixes)

### Dashboard won't open:
```bash
# Try full path:
file:///home/futurebit/.openclaw/workspace/boyd-command-center/dashboard.html
```

### Data looks old:
1. Click **Refresh** button (top right)
2. Check "Last updated" timestamp
3. If still old, tell Maxi: "Update my dashboard"

### Lost data:
1. Check if you have export backup
2. Click **Import** button
3. Select backup JSON file
4. Data restored!

### Mobile version weird:
- Rotate to landscape if needed
- Pinch to zoom out slightly
- Portrait mode works best for scrolling

## 📤 Backup Strategy (Weekly)

### Every Friday:
1. Open dashboard
2. Click **Export** button
3. File downloads: `boyd-command-center-YYYY-MM-DD.json`
4. Save to cloud (Dropbox/Google Drive)

**Why?** If browser cache clears, you can restore from backup.
