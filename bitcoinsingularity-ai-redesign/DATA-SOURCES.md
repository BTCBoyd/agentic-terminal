## Methodology Notes

**Academic Rigor:**
- Transparent data sources (cite everything)
- Conservative estimates (avoid hype)
- Acknowledge data limitations (Lightning privacy)
- Peer review (have Boyd + others validate)

**Bias Mitigation:**
- Don't just look for confirming evidence
- Document failures/challenges too
- Include skeptical perspectives
- Compare to alternative payment rails (if data available)

**Ethical Considerations:**
- Don't dox private AI agents without consent
- Respect Lightning privacy features
- Only use public/permissioned data
- Clear opt-in for case studies
