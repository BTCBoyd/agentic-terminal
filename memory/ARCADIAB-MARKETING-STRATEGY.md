# ArcadiaB Go-To-Market Strategy
**Date:** February 2026  
**Status:** Draft (not yet approved by executive team, good starting point)

## Executive Takeaway

**Key Insight:** The infrastructure is built. The constraint is now distribution to a segment (Mexican HNWIs) that doesn't convert through social media.

**Core Strategy:**
- Social media = air cover
- In-person events + referral flywheel = ground troops

**Priority Sequence:**
1. Ship new app/web
2. Nail Punta Mita as 'ArcadiaB 2.0' launch moment
3. Activate referral flywheel
4. Begin wealth manager outreach with Capital Duro webinars
5. Let Maxi run the content engine while Boyd focuses on high-value relationships
