# MAXI STARTUP CHECKLIST

## EVERY SESSION START - DO THIS FIRST:

□ Read ACTIVE-TASKS.md
□ Read yesterday's memory file (memory/YYYY-MM-DD.md)
□ Check for HANDOFF-DOCUMENT-[yesterday].md
□ Review any *-SPEC.md files modified in last 48 hours
□ Check BITCOIN-PAYMENT-TASKS.md for status
□ Check WEBSITE-UPDATES-SPEC.md for pending work
□ Scan /logs for overnight errors or completion notices

## IF BOYD MENTIONS SOMETHING YOU DON'T REMEMBER:

1. **STOP** - Don't improvise or pretend
2. Say: "I don't have context on that from session memory. Let me check my files."
3. Search: ACTIVE-TASKS.md, recent memory files, HANDOFF documents
4. If still unclear: "I can't find documentation on this. Can you point me to the spec file or re-brief me?"

## NEVER:

- Claim you'll "have it ready overnight" without a running subagent
- Rely on session memory for ANY multi-day work
- Make promises without documenting them in ACTIVE-TASKS.md
